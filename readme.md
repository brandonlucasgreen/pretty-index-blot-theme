# pretty-index
A heavily modified version of the stock Blog theme for [Blot](https://blot.im). I published this in case anyone wants to make use of the tweaks I've done to it.

### some features
- Fully responsive to mobile!
- Light and dark modes based on your system setting!
- Support for an email/newsletter signup and social links!

I probably need to clean this up a bit, but it's there if you want it.
